# Nike Brand Page

A modern Nike landing page built using React and Vite.  
This project showcases a clean UI design with a professional layout inspired by Nike branding.

## 🚀 Features

- Modern UI/UX
- React Components Structure
- Fast Development with Vite
- Clean and Reusable Code
- Stylish Hero Section

## 🛠️ Technologies Used

- React.js
- Vite
- JavaScript (ES6+)
- CSS3
